## Requirements

* NodeJS V20+
* FFMPEG
* Imagemagick
* NodeJS basic knowledges (dont be an idiot)

#### Setup bot on your Machine
```bash
$ git clone https://github.com/ammaricanooo/NexureMD-V2
$ cd NexureMD-V2
$ npm i
```

---------

## Run

```bash
npm start
```
